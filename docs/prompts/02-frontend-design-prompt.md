# Промпт: Frontend Design Skill

**Когда использовать**: Когда нужно создать фронтенд с нетривиальным визуальным стилем — не generic «AI-дефолт» с Inter и фиолетовыми градиентами. Скилл `frontend-design` переключает AI в режим арт-директора.

**Суть навыка**: Перед написанием кода AI выбирает эстетическое направление (editorial, brutalist, organic, retro-futuristic и т.д.) и только потом реализует его с конкретными шрифтами, цветами и анимациями.

---

## Промпт

```
Сделай весь фронт через `frontend-design` skill.

Продукт: [ЧТО ЭТО ЗА СЕРВИС — одно предложение]
Аудитория: [кто пользователи, их стиль жизни]
Тон: [примеры: "бережный литературный журнал", "не корпорейт и не велнес", "как Notion, только теплее"]
Стек: [React + Vite / plain HTML + CSS / Vue]

Запрещено:
- Inter, Arial, Roboto, system fonts
- Фиолетовые градиенты на белом
- Стандартные shadcn/tailwind компоненты без кастомизации

Страницы/компоненты: [список того, что нужно]
```

---

## Что AI делает с этим промптом

1. **Выбирает aesthetic direction** — одно чёткое слово или фраза: "editorial organic", "warm brutalist", "quiet luxury".
2. **Подбирает типографику**: display-шрифт для заголовков + body-шрифт. Никогда не дефолтные.
3. **Формирует палитру**: dominant + accent + neutral, через CSS-переменные.
4. **Добавляет атмосферу**: grain texture, subtle gradients, micro-animations.
5. **Пишет production-ready код** с этим стилем.

---

## Пример из MindMatch

**Мой промпт**:
> «Делай весь фронт через `frontend-design` skill. Сервис для коучей личностного роста — нужен бережный, литературный тон, не корпорейт и не "велнес".»

**Что выбрал AI**:
- **Aesthetic**: editorial-organic
- **Шрифты**: Fraunces (variable serif) для display + DM Sans для body
- **Палитра**: тёплый крем `#F4EFE6` / тёмный шоколад `#1F1B16` / акцент шафран `#D67340` / шалфей `#6B7A65`
- **Текстура**: subtile CSS grain через layered radial-gradients на `::before`
- **Орнамент**: концентрические круги SVG в hero — «символ слоёв запроса»

**Результат**: не похоже ни на один другой booking SaaS.

## Как уточнять после первого ответа

Следующий промпт-уточнение работает через конкретные пожелания к визуалу:

```
Добавь [ЭЛЕМЕНТ] в [МЕСТО] — [ОПИСАНИЕ ЧЕРЕЗ МЕТАФОРУ ИЛИ ЭМОЦИЮ].
Например: «Сделай орнамент в правом верхнем углу hero — концентрические круги тонкой линией, как символ слоёв запроса.»
```

**Почему через метафору**: AI делает лучший выбор реализации, чем когда ты диктуешь CSS. «Символ слоёв» → SVG-орнамент. «Бумажная текстура» → CSS grain. Дай ему интерпретировать.

---

## Почему это важно для студентов курса

Большинство AI-интерфейсов выглядят одинаково, потому что промпт был «сделай красивый UI». `frontend-design` skill задаёт структуру: сначала концепция, потом код. Разница между «AI-слоп» и «дизайн с точкой зрения».
